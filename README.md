# kanapoo-web

西毛エリア（高崎・前橋・藤岡）を中心に活動するライフデザイナー「かなプー」の公式サイトです。

## 概要
住友生命2年目として新たな挑戦を続ける「かなプー」のブランディングと、地域の方々との信頼を深めるための拠点となるサイトです。
プロフェッショナルとしての「知性と気品」 と、愛犬や家族を大切にする「等身大な母としての顔」 をバランスよく表現しています。

## 主な機能
* **AIかなプー（Gemini 1.5 Flash 連携）**: かなプーの分身AIが、ライフプランの相談から「万福苑」のおすすめメニューまで、24時間リアルタイムで対応します。
* **マルチコンタクト導線**: 電話・LINE・メールの3つの窓口を統合し、ユーザーが最も使いやすい方法で直接本人へ繋がることができます。
* **モバイル最適化**: スマートフォンでの閲覧を前提としたレスポンシブデザインを採用しています。

## サイト構成
* **Hero Section**: 信頼を感じさせる知的なビジュアルとキャッチコピー。
* **About Section**: 2年目の情熱と、一人の母として家族の未来を守る決意。
* **Hobbies Section**: 海鮮、白ワイン、そして「焼肉飯店 万福苑」。彼女を形作る至福のペアリング。
* **Family Section**: 癒やしと元気をくれる大切な相棒（愛犬）の紹介。
* **Contact Section**: 西毛エリア（高崎・前橋・藤岡）全域をサポートする相談窓口。

## 技術仕様
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
* **AI Engine**: Gemini 1.5 Flash (Google Generative AI SDK)
* **Hosting**: GitHub Pages
